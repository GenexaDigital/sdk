# Genexa

**GENEXA** is Neura’s core lending and borrowing protocol powered by a **vote-escrow governance mechanism** designed to dynamically control interest rates and optimize liquidity efficiency across the ecosystem.

Genexa enables users, liquidity providers, and governance participants to interact with a decentralized credit market where incentives, interest rates, and capital allocation are governed transparently by token holders.

## 🚀 Overview

Genexa is built to power the financial layer of the **Neura ecosystem** by providing a decentralized lending and borrowing infrastructure.

The protocol introduces a **vote-escrow governance system** where token holders can lock their tokens to gain voting power and influence key economic parameters such as:

* Interest rate curves
* Liquidity incentives
* Collateral parameters
* Risk configurations

This model ensures that governance participants are aligned with the long-term stability and sustainability of the protocol.

---

## ⚙️ Core Features

### 1. Decentralized Lending

Users can supply assets into liquidity pools to earn interest from borrowers.

### 2. Collateralized Borrowing

Borrowers can lock approved collateral assets and borrow supported tokens against them.

### 3. Vote-Escrow Governance

Token holders can lock governance tokens to receive **veTokens**, enabling them to vote on protocol decisions.

### 4. Dynamic Interest Rates

Interest rates adapt dynamically based on supply-demand mechanics and governance decisions.

### 5. Liquidity Incentives

Liquidity providers receive rewards for supplying assets to the protocol.

---

## 🧠 Governance Model

Genexa utilizes a **vote-escrow (veToken) system** which works as follows:

1. Users lock governance tokens for a chosen duration.
2. Locked tokens generate **veTokens** (voting power).
3. veToken holders can vote on protocol parameters.
4. Longer lock periods grant higher voting influence.

This design encourages long-term commitment to the ecosystem.
